Polski:
Jest to program, który umożliwia poruszanie się postacią, którą zdefiniowaliśmy w formie tekstu w pliku tekstowym. Sterowanie odbywa się poprzez klawisze: w, a, s, d oraz j (skok).

English:
This is a program that allows movement of a character defined in text form in a text file. Control is performed using the keys: w, a, s, d, and j (jump).
