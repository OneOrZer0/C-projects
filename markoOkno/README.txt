Polski:
Jest to program typu keylogger, który symuluje okno potwierdzenia uprawnień do kontynuacji dalszych operacji. Wprowadzony ciąg znaków jest zapisywany do pliku tekstowego w katalogu programu.

English:
This is a keylogger program that simulates a confirmation window for permissions to proceed with further operations. The entered string of characters is saved to a text file in the program directory.
